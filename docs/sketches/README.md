# Origin sketches

Hand-drawn design notes from the planning phase, kept for provenance.
The current architecture (MuJoCo + Stable-Baselines3 on cheap target
hardware: ESP32 + Raspberry Pi) traces back to these sketches.

## Files

- [`2026-05-05-stack-and-bom.pdf`](2026-05-05-stack-and-bom.pdf) —
  handwritten stack and BOM brainstorm sketched the day before
  kickoff: MuJoCo lineage (Todorov / DeepMind / Multi-Joint Contact),
  RPi / ESP32 / Jetson Orin cost comparison, SB3 + policy
  optimization, swarm-aware leg kinematics. Drawn by hand to give
  participants a sense of how the architecture was actually decided —
  not from a spec doc but from a sketchbook.
